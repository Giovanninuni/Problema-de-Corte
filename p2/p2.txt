O problema da Mochila:
Em cada iteração, ele será chamado para encontrar o melhor novo padrão de corte para adicionar à nossa solução.

O Objetivo: O Método Simplex (Algoritmo 1) nos diz que, para encontrar a melhor coluna (padrão) nova para adicionar,
precisamos resolver um subproblema. Esse subproblema é:
             maximizar g(a) = Σ λ_i.a_i

Ou seja, do Algoritmo 1:
    Capacidade da Mochila(L): É o nosso L (194).
    Peso de cada item(p_i): É o comprimeto de cada item l_i.
    Restrição de quantidade(d_i): É a demanda de cada item, d_i.
    Valor/Utilidade de cada item(v_i): É o valor λ_i(do vetor λ, ou "multiplicador Simplex").

De onde vem o λ? O Algoritmo 1 nos diz para calculá-lo resolvendo B^T.λ = c_B
    λ: Vetor utilidade que passaremos para a função da mochila

O Método (Algoritmo 2): Enumeração Implícita
    É um algoritmo de busca inteligente que evita testar todas as combinações.

Primeiro Passo: Ordenar os itens pela maior eficiência (valor/peso)
    Eficiência: θ_i = v_i/p_i = λ_i/l_i
